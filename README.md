# ComercioYA - EDA para decisiones comerciales

Este proyecto realiza un análisis exploratorio de datos (EDA) para comprender el comportamiento de clientes de e-commerce.

## Estructura
- `data/`: dataset generado (`comercioya_clientes.csv`)
- `outputs/plots/`: gráficos exportados (PNG/PDF)
- `eda_comercioya.py`: script principal
