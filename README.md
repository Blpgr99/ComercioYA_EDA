Caso de Estudio: Análisis Exploratorio de Datos – ComercioYA
Descripción del proyecto

Este proyecto consiste en un análisis exploratorio de datos (EDA) sobre el comportamiento de clientes en una plataforma de comercio electrónico ficticia llamada ComercioYA.
El objetivo fue identificar patrones de compra, hábitos de consumo y factores asociados a la satisfacción del cliente.

Desafío principal

El principal desafío fue transformar un conjunto de datos crudos en información útil para la toma de decisiones, considerando:

Datos heterogéneos (compras, visitas, devoluciones, reseñas)
Identificación de variables relevantes
Generación de insights claros para negocio
Solución propuesta

Se desarrolló un análisis estructurado que incluyó:

Limpieza y transformación de datos
Análisis univariado y bivariado
Visualización de patrones clave
Construcción de insights accionables orientados a negocio
🛠️ Herramientas utilizadas 

Python 
Pandas (manipulación de datos)
Matplotlib / Seaborn (visualización)
Jupyter Notebook

Principales aprendizajes
Importancia de la calidad de datos en el análisis
Cómo identificar variables clave para el negocio
Uso de visualizaciones para comunicar insights
Relación entre comportamiento del cliente y métricas comerciales

Métricas de impacto
Identificación de patrones de compra relevantes
Detección de factores asociados a devoluciones
Segmentación básica de clientes según comportamiento
Generación de insights para mejorar la toma de decisiones comerciales

Habilidades técnicas aplicadas
Análisis exploratorio de datos (EDA)
Limpieza y transformación de datos
Visualización de datos
Pensamiento analítico orientado a negocio
Storytelling con datos
